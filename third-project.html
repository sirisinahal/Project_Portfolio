Examining the SkillsFuture website:

The Ministry of Education established SkillsFuture Singapore (SSG) as a statutory body (MOE). It will lead and manage the national SkillsFuture initiative, support the development and comprehensive structure of learning and development based on the pursuance of skills excellence and enhance Singapore's research and skills environment.

AWS Services:

These services were chosen because they are extremely flexible, persistent, secure and stable and they can efficiently handle clustering.

API Gateway:

The Amazon API Gateway service allows you to build, expose, administer, analyse, and secure REST, HTTP, and WebSocket APIs at any scope. API developers may construct APIs that connect to AWS or other online services, along with datasets in the Aws Environment.

Lambda:

AWS Lambda is a serverless computing service that automatically maintains the associated application components while running code in accordance to events.

S3:

The Amazon Simple Storage Service (Amazon S3) is an online storage service. Amazon S3 may be used to host and get any volume of information from any location on the internet at any time.

Work Description

Here, I explored AWS services such as API Gateway, Lambda, and S3.
I also uploaded the files to an S3 bucket, wrote code to create API calls for both skills and job titles using the POST method, and gave organisational teams permission to use these API calls to retrieve the needed data using API Gateway.
For any error codes issued while accessing API calls using the POSTMAN tool, I examined and altered the code as needed.

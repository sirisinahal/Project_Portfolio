Selenium web scraping

There are other online scraping technologies available, but I picked Selenium over others since it is an automated testing framework for online applications/websites that can also manage the browser to traverse the website like a human. To trigger desired events, Selenium employs a web-driver package that may take control of the browser and simulate user-oriented activities.

Analysis of the O*NET website

The O*NET database contains a large number of variables that represent job and worker characteristics, such as skill needs. The professions are defined by the O*NET-SOC taxonomy, and the Content Model specifies which information is gathered. With data from the working public, the Data Collection programme brings these frameworks to life.

Work Description

It provided an excellent gaming experience. In this case, we had to research the O*NET website and design an uniform schema for the Skills Taxonomy. The website included substantial data in the form of 38 files, for which we discovered the linkage and constructed a schema using Python utilities . At first, analysing the data on the website caused confusion; however, after much effort, we were able to link the data threads. Finally, we gathered 45,000 technical and non-technical skills from the website.

Examining the NESTA website:

Nesta is a data-driven Skill Taxonomy that uses real world data from job listings. Many other existing skill taxonomies, such as O*Net and ESCO, depends on experienced professional consultation to some extent, which implies that amending the taxonomies may be a time-consuming and expensive procedure.
A data-driven taxonomy, such as Nesta, on the other hand, is much straightforward to modify

Work Description

I focused on enhancing the algorithm's accuracy for this Sprint.
For this, I used the BERT algorithm on a test data set of roughly 1000 job titles, before and after cleaning the data, and manually reviewed the results, calculating accuracy, precision, and frequency using the confusion matrix.
These findings lead us to conclude that the True positives and False positives data for the manually cleaned and uncleaned data had changed.
Cleaning the data manually and identifying patterns to eliminate irrelevant data is a time-consuming process.
